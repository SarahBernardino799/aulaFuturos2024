Estrutura HTML

o documento html se inicia com uma declaração que indica a versão HTML
sendo por exemplo !DOCTYPE html

Elemento HTML

contem a raiz do documento HTML, também inclui o idioma da página html lang="pt-br"

Elemento head e body

O elemento Head contém informações cruciais sobre o documento como title,link,type,media,style e script e etc.
como no exemplo abaixo.
"meta http-equiv='X-UA-Compatible' content='IE=edge' "
"title aula 01 estrutura e comentários /title"
"link rel='stylesheet' type='text/css' media='screen' href='main.css'
"script src='main.js'/script"

O elemento Body contém todo o conteúdo visível da página, como texto, imagens, vídeos, formulários, etc.
EXEMPLO
"< h 2 > a tag para adicionar o titulo < / h >"
"< p > paragrafo com o texto < / p >"

Opções de textos

Temos varias formas de de destacar os textos
para deixar em negrito usamos a palavra reservada strong
para deixar em italico usamos a palavra reservada em ou i
para deixar em sublinhado usamos a palavra reservada u
